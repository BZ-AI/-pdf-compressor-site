# PDF压缩网站

一个免费的在线PDF压缩工具，无需登录即可使用。

## 功能特点

- 在线PDF压缩
- 无需登录
- 支持拖拽上传
- 自动添加水印
- 多种压缩质量选择

## 技术栈

- Node.js + Express
- PDF处理库：pdf-lib, pdf-compressor
- 前端：HTML5 + Tailwind CSS

## 本地开发

1. 安装依赖
```bash
npm install
```

2. 启动服务器
```bash
npm start
```

3. 访问 http://localhost:3000
